Project Scope

A decision-support system for students, faculty, and advisors.

01

Profile Analysis

Reads academic history, skills, interests, experience, learning time, and career priority.

02

Interest Discovery

Maps profile signals to career tracks such as healthcare, AI, data, web, cloud, cybersecurity, and product.

03

Course Recommendation

Suggests course paths based on readiness, target role, skill gaps, and learning style.

04

Explainable Output

Shows why courses were recommended through transparent contribution factors.

Framework

From student details to explainable, fair recommendations.

1

Profile Input

Collect academic records, subjects, skills, interests, projects, and career goals.

2

Signal Scoring

Convert profile text and selected skills into interpretable career-track scores.

3

XAI Layer

Explain which skills, interests, and academic signals influenced the result.

4

Fairness Review

Keep sensitive attributes outside the scoring path and show advisor review notes.

5

Course Planner

Rank courses, skill gaps, career roles, and next learning actions.

Fairness and Explainability

Built for responsible academic decision support.

The system is designed as an advisor-support tool, not an automatic judgment. It should explain recommendations, avoid sensitive attribute scoring, and allow students or advisors to review the final path.
